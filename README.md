# Customer Segmentation Using K-Means Clustering

## Project Overview

This project applies K-Means Clustering to segment mall customers based on their Annual Income and Spending Score. The goal is to identify distinct customer groups and gain insights that can support business marketing strategies.

1. Data Loading and Exploration
2. Feature Selection
3. Elbow Method
4. K-Means Clustering
5. Cluster Visualization
6. Customer Segmentation Analysis

## Results

* Determined the optimal number of clusters using the Elbow Method.
* Segmented customers into distinct groups.
* Visualized customer clusters and cluster centroids.
* Derived business insights from customer behavior patterns.
